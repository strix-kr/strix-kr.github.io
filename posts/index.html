Hugo 로 github.io 블로그 만들기

1. 준비

1.1 저장소

  • github.com repo 2개
    • /blog : 블로그 관련 파일 관리용. 비공개
    • /strix-kr.github.io : 실제 퍼블리싱 파일들 위치. 공개

1.2 로컬 설정

  • Hugo 설치

    (mac) brew install hugo

  • 블로그 생성

    hugo new site blog && cd blog

  • 테마 설치

    git clone https://github.com/dplesca/purehugo.git themes/purehugo

  • git 설정

    git init
git remote add origin git@github.com:strix-kr/blog.git
git submodule add git@github.com:strix-kr/strix-kr.github.io.git public

2. 글 생성 및 퍼블리싱

2.1 글 생성

hugo new posts/test1.md

2.2 draft 확인

hugo server -D

2.3 발행

  • 글 머릿말의 draft 항목을 false 로 변경
  • 자동 스크립트 실행
./deploy.sh '커밋 메세지'
